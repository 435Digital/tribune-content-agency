A WordPress plugin for parsing RSS feeds and creating posts from them

== Description ==
This WordPress plugin is for parsing RSS feeds and creating posts from them. Choose the author and the post type and submit!

== Installation ==
Simply unzip and upload the four-three-five-rss folder to the plugins directory of your WordPress install and activate the plugin
